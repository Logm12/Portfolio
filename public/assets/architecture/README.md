# Architecture Diagrams

Place your architecture diagram images here:

- `viettel-llm.png` - Viettel AI Coding Assistant architecture
- `recsys-graph.png` - Graph-Based Movie RecSys architecture
- `trading-mlops.png` - Real-time Algo Trading MLOps architecture
- `retention-causal.png` - Customer Retention Causal Inference architecture

## How to add:

1. Create/export your architecture diagrams as PNG or JPG
2. Name them according to the filenames above
3. Place them in this folder (`public/assets/architecture/`)
4. The portfolio will automatically display them in the project modal

## Tools to create diagrams:

- [Excalidraw](https://excalidraw.com/) - Hand-drawn style
- [draw.io](https://draw.io/) - Professional diagrams
- [Mermaid Live Editor](https://mermaid.live/) - Code-based diagrams
- [Figma](https://figma.com/) - Design tool
