# Awards & Certificates

Place your certificate/award images here:

- `viettel-2nd-place.png` - Giải Nhì Vô địch Tin học VP Thế giới - Viettel 2025
- `sinh-vien-5-tot.png` - Sinh viên 5 tốt certificate
- `gpa-transcript.png` - GPA/Bảng điểm (optional)

## How to add:

1. Scan or screenshot your certificates
2. Crop and optimize the images (recommended: 800x600 or similar aspect ratio)
3. Name them according to the filenames above
4. Place them in this folder (`public/assets/awards/`)
5. Update `src/data/projects.ts` with correct filenames

## Tips:

- Use PNG for documents with text
- Optimize file size (< 500KB recommended)
- Maintain consistent aspect ratios
